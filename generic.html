Some of My Projects

Throughout my internships and academic journey, I've had the opportunity to work on various projects that have honed my skills in web and mobile development, data engineering, and more. Below is a selection of some of my most notable work:

Data Pipeline for BotiEducation

A robust data pipeline developed during an internship, designed to parse complex datasets and load them into a BigQuery sink. This project involved extracting data from various sources, transforming it to meet analytical needs, and efficiently loading it into BigQuery for further processing and analysis. The pipeline was optimized for performance and scalability, ensuring data integrity throughout the process(Internship Project).

Sign Language Translator

Sign Language Translator (Using AI): This innovative mobile app utilizes artificial intelligence to translate sign language into text or speech, helping bridge communication gaps. Built with Flutter, the app processes real-time hand gestures and converts them into easily understandable language, offering an intuitive and accessible user experience. Whether you're learning sign language or communicating with a deaf person, this app provides seamless, instant translations, making communication more inclusive for everyone.

Movie heaven

Web App (Spring Boot & Thymeleaf): "A web-based movie streaming platform designed to provide a smooth and interactive user experience. Developed using Spring Boot and Thymeleaf, this app integrates multiple APIs to offer a comprehensive movie library. Users can easily navigate, search, and watch movies directly from their browsers, all wrapped in a clean and modern design.
